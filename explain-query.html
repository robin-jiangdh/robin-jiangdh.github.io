EXPLAIN query

#EXPLAIN query linkTitle: "EXPLAIN query" description: > EXPLAIN query

EXPLAIN types

EXPLAIN AST SYNTAX PLAN header = 0, description = 1, actions = 0, optimize = 1 PIPELINE header = 0, graph = 0, compact = 1 ESTIMATE SELECT ...

  • AST - abstract syntax tree

  • SYNTAX - query text after AST-level optimizations

  • PLAN - query execution plan

  • PIPELINE - query execution pipeline

  • ESTIMATE - https://github.com/ClickHouse/ClickHouse/pull/26131 (since 21.9)
